# Map render with A star (Процедурная генерация с помощью A star)

## Setup (Установка)

npm install

## Run (Запуск)

**npm run start** or (или) **node app.js**

Map 60 x 30 blocks (Карта 80 x 40 блоков)
white block - wall (белый блок - стена)
grey block - floor (сервый блок - пол)
